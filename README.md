# napoleonx-ens-datachallenge-crypto-forecasting-2025

## Contexte du projet

Ce projet a été réalisé dans le cadre du **Challenge Data ENS**, proposé par l’entreprise **Napoleon X**.

## Objectif 

L’objectif est de **prédire le rendement relatif au Bitcoin de clusters de cryptomonnaies**, à partir de données historiques de rendements horaires et d’informations sur la structure des clusters.

## Datasets

Les datasets utilisés dans ce projet sont disponibles au téléchargement sur ce lien : https://challengedata.ens.fr/participants/challenges/71/
Il faut créer un compte participant pour pouvoir les télécharger.
